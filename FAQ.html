What version of Mediaroom does JackFish support?

JackFish plant has been tested with Mediaroom 1.1 and 2.0. It removed call to all depreciated API's as listed by Microsoft API documentation 2.0. JackFish Billing Adapter used Mediaroom 2.0 DataWare House API's to pull Billing information. If JackFish Billing adapter detects unavailability of DataWare House, it used Reporting Store API to pull Billing Records.

What is Hardware specification requirement?

JackFish Plant have been tested on Windows Server 2008 R2 / 2012. BOM of JackFish suit required 2 Load Balanced machines for Framework 2 Load Balanced Machines for Database 2 Load Balanced Machines to host UI Application.

Does JackFish maintain separate copy of Mediaroom Data?

JackFish do maintain a separate copy of database which is in sync with Mediaroom using scheduled jobs. It keeps the local copy of accounts/devices/groups/services/service collections(PPV/LIVE/VOD/RDP)/packages/grants/EPG Data. It keeps all the information about subscribers there packages, EPG data with call letters, Channel Maps. All these data are exposed through services which can be consumed by operators.

Does JackFish Support extracting and archiving Billing Records?

Billing Module does currently support importing billing record, reconciling with PPV and VOD assets to pull the title and rating. Once the records are processed (exported in appropriate format) it marks the status as readable. Billing module support Archive using Billing Achieve job which archive the old record (age defined by configuration) and mark the status of records to Delete in Mediaroom after successfully archiving the records. It creates achieve MDF files which can be restored any time in future if any requirement to view the archived Billing records is provided.

What types of reports are supported by JackFish Plant?

JackFish provide following report out of box. JackFish supports any custom reports requirements.

  • User Report - general user information, related vendor and group details as well as status and security details.
  • Subscriber Class Report - displays the subscribers associated to each subscriber class and related subscriber details
  • Credit Limit Reset - provides information about the batch credit limit updates processing. It provides information on the credit limit reset updates by VHO and by date for the updates that were not successful
  • Audit Event - provides an overall resource usage information, including measure of activity (successes and failures) within the SDP for each user (id) after a login. An audit trail generates important data related to user and system activities during normal system operation for a given time period.
  • Live Channel Subscription - displays the number of active Live Channel subscriptions held by subscribers, on a particular date. The listing displays active subscription count per live channel on a per day basis
  • VoD/PPV Purchase Settlement - displays the number of daily purchases of assets (VoDs and PPVs) by subscribers on a particular date. The listing displays the purchase counts by date and by price

How does JackFish supports subscriber privacy?

All JackFish products use machine key to encrypt the sensitive data i.e. service credential for web services. These machine keys are generated specific for machine and hence the data can’t be unencrypted in any other machine. User name password are not stored. They are stored as hash and hence they can’t be read by any other third party. In future if any other sensitive data is stored, it will be encrypted with this machine key.

What security model JackFish provide??

JackFish support role based security for all operations exposed as Service. Below are the role and current function that they are authorized to access.

Role Name Role Description
Subscriber Can perform end-user actions
ContentProvider Can audit content (live, VOD, PPV, application) usage
Installer Can install devices and associate with accounts
CSR Can modify carrier-specific principals and perform device-level actions
Operator Can modify carrier-specific principals and channel maps
CarrierManager Can modify all carriers, principals, and channel maps
Administrator Can manage every aspect of the system, including configure the JackFish Framework and modules
